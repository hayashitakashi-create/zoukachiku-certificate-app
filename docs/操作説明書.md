## 1. はじめに

本アプリケーションは、増改築等工事証明書を効率的に作成・管理するためのWebアプリです。

### アクセスURL

**https://zoukachiku-certificate-app.vercel.app**

### 対応する証明書の種類

| 種別 | 名称 | 様式ページ |
|------|------|-----------|
| 住宅ローン減税 | 住宅借入金等特別税額控除 | 様式 1ページ |
| リフォーム促進税制 | 住宅耐震改修特別税額控除 又は 住宅特定改修特別税額控除 | 様式 9ページ |
| 買取再販 | 買取再販住宅の取得に係る住宅借入金等特別税額控除 | 様式 17ページ |
| 固定資産税 | 固定資産税の減額に資する改修工事 | 様式 20ページ |

### 対応する工事種別

| 工事種別 | 説明 |
|---------|------|
| 耐震改修工事 | 住宅の耐震性を向上させる工事 |
| バリアフリー改修工事 | 高齢者等の居住改善のための工事 |
| 省エネ改修工事 | 断熱性・省エネ性能を向上させる工事 |
| 同居対応改修工事 | 多世代同居に対応するための工事 |
| 子育て対応改修工事 | 子育て環境を改善するための工事 |
| 耐久性向上改修工事 | 長期優良住宅化のための工事 |
| その他増改築等工事 | 上記に該当しないその他の工事 |

### データ保存について

| 保存先 | 対象データ | 条件 |
|--------|-----------|------|
| ブラウザ内（IndexedDB） | 証明書データ | 常時 |
| サーバー（PostgreSQL） | ユーザーアカウント・発行者設定 | ログイン時 |

証明書データはブラウザ内に保存されるため、定期的なバックアップを推奨します（→ [10. データのバックアップ・復元](#10-データのバックアップ復元)）。

### 対応デバイス

- PC（デスクトップ）
- タブレット
- スマートフォン（iPhone SE 375px〜対応）

## 2. ユーザー登録・ログイン

### 2-1. 新規登録

1. ログインページ下部の「**ユーザー登録**」リンクをクリック
2. 以下の情報を入力

| 項目 | 必須 | 説明 |
|------|------|------|
| 氏名 | - | ユーザーの表示名 |
| メールアドレス | ○ | ログインに使用 |
| パスワード | ○ | 8文字以上 |
| パスワード（確認） | ○ | 再入力して確認 |

3. 「**ユーザー登録**」ボタンをクリック
4. 登録成功後、自動的にログインしてトップページへ遷移

### 2-2. ログイン

1. メールアドレスとパスワードを入力
2. 「**ログイン**」ボタンをクリック
3. ログイン成功後、トップページへ遷移

## 4. 証明書の新規作成（6ステップウィザード）

トップページの「**新しい証明書を作成**」をクリックして開始します。

画面上部のステッパーで現在の進捗を確認できます。

### ステップ1: 基本情報

以下の情報を入力します。

**申請者情報**

| 項目 | 必須 | 説明 |
|------|------|------|
| 氏名 | ○ | 証明書の申請者名 |
| 郵便番号 | - | 入力すると住所が自動補完されます |
| 住所 | ○ | 申請者の住所 |
| 番地・建物名 | - | 建物名や部屋番号 |

**家屋番号及び所在地**

| 項目 | 必須 | 説明 |
|------|------|------|
| 郵便番号 | - | 入力すると所在地が自動補完されます |
| 所在地 | ○ | 工事対象の家屋の所在地 |
| 家屋番号 | - | 登記簿上の家屋番号 |

**証明書の用途**

以下の4つから1つを選択します：

- **Ⅰ．所得税額の特別控除**
  - 住宅ローン減税（増改築）
  - リフォーム促進税制
  - 買取再販住宅
- **Ⅱ．固定資産税の減額**
  - 固定資産税減額

**工事情報**

| 項目 | 必須 | 説明 |
|------|------|------|
| 工事完了年月日 | ○ | 工事が完了した日付 |

入力が終わったら「**次へ**」をクリックします。

> 入力データは自動保存されます。ブラウザを閉じても、次回アクセス時に続きから入力できます。

### ステップ2: （1）実施した工事の種別

選択した証明書の用途に応じて、工事種別の詳細項目を入力します。

**住宅ローン減税の場合**
- 第1号〜第6号工事の詳細チェック項目
- 住宅ローン詳細ページへのリンク

**リフォーム促進税制の場合**
- ① 住宅耐震改修
- ② バリアフリー改修工事
- ③ 省エネ改修工事（窓断熱・低炭素建築物・設備型式・追加工事）
- ④ 同居対応改修工事（改修前後の室数）
- ⑤ 耐久性向上改修工事（11項目＋認定情報）
- ⑥ 子育て対応改修工事（6項目）
- 併せて行う第1号〜第6号工事

**固定資産税の場合**
- 1-1. 耐震改修
- 1-2. 耐震改修→認定長期優良住宅
- 2. 熱損失防止改修工事等（省エネ）

> 該当しない項目は空欄のままで構いません。

### ステップ3: （2）実施した工事の内容

各工事種別について、実施した工事の内容をテキストで記述します。

### ステップ4: （3）費用の額等

選択した用途に応じて、工事費用の詳細を入力します。計算ページで入力した工事データがある場合は自動で反映されます。

**共通項目**
- ア: 工事費用の合計額
- イ: 補助金等の有無と金額
- ウ: 差引額（自動計算）
- エ: 控除限度額との比較（自動計算）
- オ: 超過額（自動計算）

**リフォーム促進税制の場合**
- 各工事種別ごとに費用を入力
- ⑤ 耐久性向上（OR型）: ベース改修＋耐久性向上の2段構造
- ⑥ 耐久性向上（AND型）: 耐震＋省エネ＋耐久性向上の3段構造
- パターン比較（⑧〜⑲）が自動計算されます

> 50万円超の要件チェック、控除上限額の適用など、公式様式に準拠した自動計算が行われます。

### ステップ5: 証明者情報

証明書の発行者情報を入力します。

| 項目 | 必須 | 説明 |
|------|------|------|
| 組織区分 | ○ | 登録建築士事務所／指定確認検査機関 等 |
| 建築士氏名 | ○ | 証明を行う建築士の氏名 |
| 資格区分 | ○ | 一級建築士／二級建築士／木造建築士 |
| 登録番号 | ○ | 建築士登録番号 |
| 事務所名 | ○ | 所属する事務所名 |
| 事務所住所 | ○ | 事務所の所在地 |
| 発行日 | ○ | 証明書の発行日 |

> 設定ページで発行者情報を事前登録しておくと、自動で入力されます（→ [9. 発行者情報の設定](#9-発行者情報の設定)）。

### ステップ6: 確認・保存

入力した内容が一覧表示されます。各セクションの「編集」リンクから該当ステップに戻って修正できます。

内容を確認し、以下のいずれかで保存します。

| ボタン | 説明 |
|--------|------|
| **下書き保存** | 下書きとして保存。後から編集可能 |
| **保存して完了** | 完了状態で保存。PDF出力が可能になります |

保存後、証明書の詳細ページに遷移します。

## 5. 工事データの入力（計算ページ）

証明書の詳細ページから、各工事種別の計算ページにアクセスします。

### 対象ページ一覧

| ページ | 対応する工事 | URL |
|--------|-------------|-----|
| 耐震改修工事 | 住宅の耐震性を向上させる工事 | /seismic-reform |
| バリアフリー改修工事 | 高齢者等の居住改善のための工事 | /barrier-free-reform |
| 省エネ改修工事 | 断熱性・省エネ性能を向上させる工事 | /energy-saving-reform |
| 同居対応改修工事 | 多世代同居に対応するための工事 | /cohabitation-reform |
| 子育て対応改修工事 | 子育て環境を改善するための工事 | /childcare-reform |
| 耐久性向上改修工事 | 長期優良住宅化のための工事 | /long-term-housing |
| その他増改築等工事 | 上記に該当しないその他の工事 | /other-renovation |

### 共通の操作手順

1. **証明書の選択**: 対象の証明書が自動で読み込まれます（証明書情報がページ上部に表示されます）
2. **工事項目の追加**:
   - 工事種別コードを選択
   - 単価・数量・居住割合を入力
   - 「**追加**」をクリック
   - 計算額（単価 × 数量 × 居住割合）が自動計算されます
3. **登録済み項目の確認**: テーブルに追加した工事項目が一覧表示されます
4. **項目の編集・削除**: 各行の「編集」「削除」ボタンで操作
5. **サマリーの確認**:
   - 合計額
   - 補助金額
   - 控除対象額
   - 控除上限額との比較
6. 「**保存**」をクリックして証明書に反映

### 工事種別ごとの特記事項

**省エネ改修工事**
- 窓面積比率（0〜100%）の入力が必要な場合があります
- 太陽光発電設備がある場合、控除上限額が引き上げられます（250万円→350万円）

**耐久性向上改修工事**
- 「認定長期優良住宅」にチェックすると、控除上限額が引き上げられます（250万円→500万円/600万円）

**その他増改築等工事**
- 単価×数量ではなく、金額を直接入力します
- 控除上限額の制限はありません

## 6. 住宅ローン詳細ページ

住宅ローン減税を選択した場合に使用する専用ページです。

### アクセス方法

証明書作成ウィザードのステップ2、または証明書詳細ページからアクセスします。

### 入力内容

**（1）工事の種別**

6つの工事号に分類された詳細項目にチェックを入れます。

| 号 | 内容 |
|----|------|
| 第1号工事 | 増築・改築・大規模の修繕・大規模の模様替 |
| 第2号工事 | 床・階段・間仕切壁・主要構造部である壁の過半修繕 |
| 第3号工事 | 居室・調理室・浴室・便所・洗面所・納戸・玄関・廊下の修繕 |
| 第4号工事 | 耐震性向上（建築基準法準拠・地震安全性基準） |
| 第5号工事 | バリアフリー改修（通路拡幅・手すり・段差解消 等） |
| 第6号工事 | 省エネ改修（窓断熱・天井壁床断熱・設備設置 等） |

**（2）工事の内容**

実施した工事の具体的な内容を記述します。

**（3）費用の額等**

| 項目 | 説明 |
|------|------|
| ① 工事費用の合計額 | 第1号〜第6号工事の合計 |
| ② 補助金等 | 有無の選択と金額 |
| 控除対象額 | ①−② が自動計算（100万円以上必要） |

> 控除対象額が100万円未満の場合、警告メッセージが表示されます。

入力後「**保存**」をクリックして、証明書に反映します。

## 7. 証明書の詳細・管理

### 詳細ページの構成

トップページで証明書カードをクリックすると、詳細ページが表示されます。

**表示される情報**
- 基本情報（申請者名・住所・物件所在地・家屋番号・工事完了日）
- 証明書の用途区分（住宅ローン減税の場合は費用概要も表示）
- 発行者情報（組織・氏名・資格・事務所名・発行日）
- 登録済みの工事データ（種別ごとの件数・合計額・控除対象額）

### 操作ボタン

ページ上部のヘッダーにアクションボタンが並んでいます。

| ボタン | 説明 |
|--------|------|
| **← 一覧へ戻る** | トップページへ戻る |
| **プレビュー** | 公式様式に準拠したレイアウトで表示 |
| **PDF出力** | PDF形式でダウンロード |
| **印刷** | ブラウザの印刷ダイアログを開く |
| **削除** | 証明書を削除（確認ダイアログあり） |

### セクションごとの編集

各情報セクション（基本情報・用途区分・証明者情報）に「**編集**」リンクがあり、クリックすると証明書作成ウィザードの該当ステップに遷移して編集できます。

### 工事データの追加・編集

詳細ページ下部の「**改修工事一覧**」セクションに、登録済みの工事データが表示されます。

- 各工事種別ごとに件数・合計額・控除対象額を表示
- 「**編集**」ボタンで該当する計算ページへ遷移
- 「**削除**」ボタンで工事データを削除
- 工事種別ボタン群から新しい工事種別の計算ページにアクセス可能

## 8. 証明書のプレビュー・PDF出力・印刷

### プレビュー

証明書詳細ページの「**プレビュー**」ボタンをクリックすると、公式の増改築等工事証明書の様式に準拠したレイアウトで表示されます。

**表示される内容**
- 基本情報（申請者・物件情報）
- （1）証明書の用途（用途ごとの詳細説明付き）
- 住宅ローン詳細（住宅ローン減税の場合）
- （3）費用の額等（工事種別ごとの計算結果・パターン比較表・控除額計算）
- 証明書発行者情報

### PDF出力

証明書詳細ページの「**PDF出力**」ボタンをクリックすると、PDFファイルが生成されダウンロードされます。

**対応するPDF様式**

| 用途 | PDF様式 |
|------|--------|
| 住宅ローン減税 | 住宅ローン減税用様式 |
| リフォーム促進税制 | リフォーム促進税制用様式 |
| 買取再販 | 買取再販住宅用様式 |
| 固定資産税 | 固定資産税用様式 |

> PDF出力は証明書のステータスが「完了」の場合に利用できます。

### 印刷

証明書詳細ページの「**印刷**」ボタンをクリックすると、ブラウザの印刷ダイアログが開きます。プレビューページからも印刷が可能です。

> 印刷時はA4サイズに最適化されたレイアウトで出力されます。

## 9. 発行者情報の設定

証明書に記載する発行者（建築士等）の情報を事前に登録しておくことで、証明書作成時に自動入力されます。

### アクセス方法

トップページ右上の**歯車アイコン（⚙）**をクリックします。

### 設定項目

**組織区分**（以下から選択）
- 登録建築士事務所
- 指定確認検査機関
- 登録住宅性能評価機関

## 10. データのバックアップ・復元

### エクスポート（バックアップ）

1. トップページ右上の**下矢印アイコン（↓）**をクリック
2. JSONファイル（`zoukachiku-backup-YYYY-MM-DD.json`）が自動ダウンロードされます
3. ファイルを安全な場所に保管してください

### インポート（復元）

1. トップページ右上の**上矢印アイコン（↑）**をクリック
2. ファイル選択ダイアログでエクスポートしたJSONファイルを選択
3. インポート結果が表示されます
   - **インポート済み**: 新規追加または更新された件数
   - **スキップ**: ローカルデータの方が新しいためスキップされた件数

### インポート時の動作

| 条件 | 動作 |
|------|------|
| ローカルに存在しないデータ | 新規追加 |
| ローカルより新しいデータ | 上書き更新 |
| ローカルの方が新しいデータ | スキップ（ローカルを維持） |

> データの削除は行われません。安全にマージされます。

## 12. 注意事項

### データ保存に関する注意

- 証明書データは**ブラウザ内（IndexedDB）**に保存されます
- 以下の場合、データが失われます：
  - ブラウザのデータ（キャッシュ・Cookie等）を削除した場合
  - 別のブラウザや端末でアクセスした場合
  - ブラウザをリセットした場合
- **定期的なエクスポート（バックアップ）を強く推奨します**
- 別端末への移行にはエクスポート→インポートをご利用ください

### 推奨環境

| ブラウザ | バージョン |
|---------|-----------|
| Google Chrome | 最新版 |
| Microsoft Edge | 最新版 |
| Firefox | 最新版 |
| Safari | 最新版 |

### 計算の正確性

- 全ての計算は公式の増改築等工事証明書の様式に準拠しています
- 控除上限額・50万円超要件等は自動でチェックされます
- 最終的な税額控除の適用可否は、管轄の税務署にご確認ください

### セキュリティ

- パスワードは暗号化して保存されます（bcrypt）
- 通信はHTTPSで暗号化されます
- セッションはJWTトークンで管理されます

## よくある質問（FAQ）

**Q: 別のPCやスマホでデータを引き継ぐには？**
A: エクスポートでJSONファイルを保存し、移行先のブラウザでインポートしてください。

**Q: 証明書のステータスを「完了」から「下書き」に戻せますか？**
A: 証明書詳細ページから編集を行い、再度「下書き保存」をすることで下書きに戻せます。

**Q: PDF出力ができません。**
A: 証明書のステータスが「完了」になっているか確認してください。下書き状態ではPDF出力できません。

**Q: 発行者情報を毎回入力するのが面倒です。**
A: 設定ページ（⚙アイコン）で発行者情報を事前登録すると、証明書作成時に自動入力されます。

**Q: 工事データの計算結果がステップ4に反映されません。**
A: 計算ページで「保存」をクリック後、証明書の編集画面で再度ステップ4を開くと反映されます。

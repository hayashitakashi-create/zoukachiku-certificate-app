## 1. はじめに

本システムは、増改築等工事証明書の作成・管理・PDF出力を行うWebアプリケーションです。

### 対応する証明書の用途

| 用途 | 概要 |
|------|------|
| 住宅借入金等特別控除 | 住宅ローン控除の適用を受けるための証明書 |
| 住宅借入金等特別税額控除 | 改修促進税制の適用を受けるための証明書 |
| 既存住宅の譲渡所得の特別控除等 | 譲渡所得控除の適用を受けるための証明書 |
| 固定資産税の減額 | 固定資産税減額措置の適用を受けるための証明書 |

### 対応する工事種別（7種）

| 工事種別 | 内容 |
|----------|------|
| 耐震改修工事 | 住宅の耐震性を高める改修 |
| バリアフリー改修工事 | 高齢者等の移動を容易にする改修 |
| 省エネ改修工事 | 省エネルギー性能を高める改修 |
| 同居対応改修工事 | 多世帯同居に必要な設備の設置 |
| 子育て対応改修工事 | 子育てしやすい環境への改修 |
| 長期優良住宅化改修工事 | 長期優良住宅の認定基準を満たす改修 |
| その他増改築等工事 | 大規模修繕・模様替え・増築等 |

### 動作環境

- Google Chrome（推奨）、Microsoft Edge、Safari 等のモダンブラウザ
- インターネット接続が必要です

## 2. ログイン・アカウント登録

### 2.1 新規アカウント登録

1. ログイン画面下部の **「アカウントを作成」** リンクをクリック
2. 以下の項目を入力

| 項目 | 必須 | 備考 |
|------|------|------|
| 名前 | - | 任意 |
| メールアドレス | 必須 | 一意である必要があります |
| パスワード | 必須 | 8文字以上 |
| パスワード（確認） | 必須 | 上記と一致すること |

3. **「アカウントを作成」** ボタンをクリック
4. 登録完了後、自動でログインされホーム画面に遷移します

### 2.2 ログイン

1. メールアドレスとパスワードを入力
2. **「ログイン」** ボタンをクリック
3. ホーム画面に遷移します

### 2.3 ログアウト

- 画面右上の **「ログアウト」** をクリック

## 4. 証明書の新規作成

ホーム画面の **「新しい証明書を作成」** ボタンから6ステップのウィザードで作成します。

### ステップ1: 基本情報

#### 申請者情報

| 項目 | 必須 | 備考 |
|------|------|------|
| 氏名 | 必須 | 証明申請者の氏名 |
| 郵便番号 | - | 7桁入力で住所を自動補完 |
| 住所 | 必須 | 証明申請者の住所 |
| 番地・建物名 | - | 住所の詳細 |

#### 家屋番号及び所在地

| 項目 | 必須 | 備考 |
|------|------|------|
| 郵便番号 | - | 7桁入力で所在地を自動補完 |
| 所在地 | 必須 | 工事対象家屋の所在地 |
| 家屋番号 | - | 登記上の家屋番号 |

#### 工事情報・用途

| 項目 | 必須 | 備考 |
|------|------|------|
| 工事完了年月日 | 必須 | 工事が完了した日付 |
| 証明書の用途 | 必須 | 4つの用途から1つ選択 |

入力後、**「次へ」** をクリックします。

> **ヒント**: 入力中のデータは自動的に下書き保存されます。ブラウザを閉じても次回アクセス時に復元されます。

### ステップ2: 工事内容の選択

実施した工事種別をチェックボックスで選択します（複数選択可）。

- 耐震改修工事
- バリアフリー改修工事
- 省エネ改修工事
- 同居対応改修工事
- 子育て対応改修工事
- その他増改築等工事
- 長期優良住宅化改修工事

### ステップ3: 費用計算

ステップ2で選択した工事種別ごとに、標準的な費用の額を計算します。

- 工事種別ごとに工事項目を追加
- 単価・数量・割合を入力すると金額が自動計算
- 補助金額を入力すると控除対象額が算出

### ステップ4: 実施した工事の内容

選択した工事種別ごとに、実施した工事の内容を自由記述で入力します。

### ステップ5: 証明者情報

証明書の発行者情報を入力します。

> **ヒント**: [設定画面](#9-設定証明者情報)であらかじめデフォルトの証明者情報を登録しておくと、このステップで自動入力されます。

| 項目 | 説明 |
|------|------|
| 組織種別 | 登録建築士事務所 / 指定確認検査機関 / 登録住宅性能評価機関 / 住宅瑕疵担保責任保険法人 から選択 |
| 建築士情報 | 氏名・資格・登録番号 |
| 事務所・機関情報 | 名称・住所・登録番号等 |
| 発行日 | 証明書の発行日（デフォルト: 当日） |

詳しい入力項目は組織種別により異なります（[証明者情報の詳細](#91-組織種別ごとの入力項目)を参照）。

### ステップ6: 確認・保存

入力内容の確認画面です。各セクションの **「編集」** リンクから該当ステップに戻れます。

| ボタン | 説明 |
|--------|------|
| **下書き保存** | 下書きとして保存（後から編集可能） |
| **保存して完了** | 完了状態で保存 |

保存後、ホーム画面に戻ります。

## 6. 工事データの入力

各工事種別の入力画面では、工事項目ごとに以下を入力します。

### 共通の入力項目

| 項目 | 説明 |
|------|------|
| 工事種別 | プルダウンから選択（単価・単位が自動設定） |
| 単価 | 標準的な費用の額（円） |
| 数量 | 施工数量 |
| 単位 | ㎡、箇所、m 等（自動設定） |
| 居住用部分の割合 | 0〜100%（居住用でない部分がある場合） |
| 補助金額 | 交付される補助金等の額 |

### 計算ロジック

```
工事金額 = 単価 × 数量 × (居住用割合 / 100)
控除対象額 = 工事金額合計 − 補助金額
```

### 工事種別ごとの注意点

| 工事種別 | 特記事項 |
|----------|----------|
| 省エネ改修 | 太陽光発電設備の有無で上限額が変動（250万円 / 350万円） |
| 長期優良住宅化改修 | 耐震+省エネの組合せにより上限額が変動（250〜600万円） |
| その他増改築等 | 金額を直接入力。上限なし |

入力後、**「保存」** ボタンで証明書に工事データを紐付けます。

## 7. 住宅借入金等特別控除の詳細入力

証明書の用途が「住宅借入金等特別控除」の場合、追加の詳細情報を入力します。

### (1) 実施した工事の種別

第1号〜第6号工事のうち、該当するものをチェックします。

| 号 | 内容 |
|----|------|
| 第1号工事 | 増築・改築・大規模修繕・大規模模様替 |
| 第2号工事 | マンション共用部分の床・階段・壁等の修繕 |
| 第3号工事 | 居室・台所・浴室等の一室全体の修繕 |
| 第4号工事 | 耐震改修工事 |
| 第5号工事 | バリアフリー改修工事 |
| 第6号工事 | 省エネ改修工事 |

第6号工事は、断熱改修の種類・地域区分・性能等級など詳細な選択項目があります。

### (2) 実施した工事の内容

工事内容の概要を自由記述で入力します。

### (3) 実施した工事の費用の概要

| 項目 | 説明 |
|------|------|
| 工事費用の額 | 第1号〜第6号工事に要した費用の合計額 |
| 補助金等の交付 | 有 / 無 を選択。「有」の場合は金額を入力 |
| 控除対象額 | 費用の額 − 補助金額（自動計算） |

> **注意**: 控除対象額が **100万円以上** でないと住宅ローン控除の適用を受けられません。画面上に判定結果が表示されます。

## 8. 証明書プレビュー・PDF出力

### プレビュー

証明書詳細画面の **「プレビュー」** ボタンで確認画面を開きます。

プレビュー画面には以下が表示されます:
- 基本情報（申請者・物件）
- 証明書の用途と制度の概要
- 住宅ローン詳細（該当する場合）
- 工事種別ごとの費用計算結果
- 計算パターン比較表（P1/P2/P3）
- 最終的な控除額
- 証明者情報

### PDF出力

証明書詳細画面の **「PDF」** ボタンをクリックすると、用途に応じたPDFファイルが自動生成されダウンロードされます。

| 用途 | PDF内容 |
|------|---------|
| 住宅借入金等特別控除 | 全工事の費用計算・ローン詳細を含む証明書 |
| 特別税額控除 | 改修促進税制用の証明書 |
| 譲渡所得控除 | 既存住宅売買用の証明書 |
| 固定資産税減額 | 固定資産税減額用の証明書 |

### 印刷

**「印刷」** ボタンでブラウザの印刷ダイアログが開きます。A4サイズで最適化されています。

## 10. エクスポート・インポート

### エクスポート（バックアップ）

1. ホーム画面の **「エクスポート」** をクリック
2. 全証明書がJSON形式でダウンロードされます
3. ファイル名: `zoukachiku-backup-YYYY-MM-DD.json`

> 定期的にバックアップを取ることを推奨します。

### インポート（復元）

1. ホーム画面の **「インポート」** をクリック
2. ファイル選択ダイアログが開きます
3. エクスポートしたJSONファイルを選択
4. 結果が表示されます: 「N件追加、M件スキップ」

| 条件 | 動作 |
|------|------|
| 新しい証明書 | 追加 |
| 既存IDと重複（既存が新しい） | スキップ |
| 既存IDと重複（インポート側が新しい） | 上書き |

## 11. 管理者機能

管理者（admin）ロールのユーザーのみアクセスできます。

### アクセス方法

URLに `/admin` を付けてアクセスします。
例: `https://zoukachiku-certificate-app.vercel.app/admin`

### 11.1 管理者ダッシュボード

統計情報をカード形式で表示します:
- 総ユーザー数
- 管理者数
- 総証明書数
- 今月の新規ユーザー数

直近の登録ユーザーと作成された証明書が一覧表示されます。

### 11.2 ユーザー管理

| 機能 | 説明 |
|------|------|
| 一覧表示 | 全ユーザーを名前・メール・ロール・登録日と共に表示 |
| 検索 | 名前またはメールアドレスで絞り込み |
| ロール変更 | 建築士 ⇔ 管理者 を切り替え |
| ユーザー削除 | ユーザーを削除（確認ダイアログあり） |

> **制限**: 自分自身のロール変更・削除はできません。

### 11.3 証明書管理

全ユーザーの証明書を横断的に閲覧できます。

| フィルタ | 選択肢 |
|----------|--------|
| テキスト検索 | 申請者名・住所で検索 |
| ステータス | すべて / 下書き / 完了 |
| 用途 | すべて / 住宅借入金等特別控除 / 特別税額控除 / 譲渡所得控除 / 固定資産税減額 |

各証明書の **「表示」** ボタンから詳細画面に遷移できます。

## 付録: よくある質問

### Q. 別のブラウザ/端末でデータは引き継げますか？

**証明者情報（設定）**: ログインすれば別の端末でも利用可能です（サーバーに保存）。

**証明書データ**: ブラウザ内（IndexedDB）に保存されています。別の端末で利用するには、エクスポート→インポートで移行してください。

### Q. データが消えてしまいました

ブラウザのデータ消去やキャッシュクリアを行うと、証明書データが失われます。定期的に **エクスポート** でバックアップを取ってください。

### Q. 本ツールの計算結果はそのまま使えますか？

本ツールの計算結果は参考値です。最終的な金額は税理士等の専門家にご確認ください。
